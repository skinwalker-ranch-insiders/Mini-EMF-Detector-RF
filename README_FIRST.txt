This project requires the Manchester libraries for Arduino Software.
I am using Arduino 1.8.57.0 from the Windows 11 App Store.
I was told the Manchester libraries do not work in the newer version of the Arduino softrware.

Download and unzip this arduino-libs-manchester-master.zip file to the follow folder:

When unzipping this zip file, extract it to your Documents folder and to the Arduino folder.
The following arduino-libs-manchester-master folder will be created with this files needed to flash the Attiny85.

arduino-libs-manchester-master.zip:
	keywords		(It gives you a list of commands that can be used when call man. commands)
	library
	Manchester.cpp
	Machester.h		(#include <Manchester.h> in the ino files)
	README.markdown
